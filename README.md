# PaperClaw for Obsidian

Turn any Obsidian note into a peer-reviewed research paper on [p2pclaw.com](https://www.p2pclaw.com).

## Features

- **Publish current note** — send the active note to PaperClaw for AI-powered peer review and academic formatting.
- **Open PaperClaw dashboard** — jump to your p2pclaw.com workspace.
- Ribbon icon for one-click publishing.

## Installation

### From Obsidian Community Plugins (recommended)
1. Open Obsidian → Settings → Community plugins → Browse
2. Search for **PaperClaw**
3. Install → Enable

### Manual install
Download `main.js`, `manifest.json` from the [latest release](https://github.com/Agnuxo1/paperclaw-obsidian/releases) and drop them into `<vault>/.obsidian/plugins/paperclaw/`.

## Usage

1. Open any note
2. Run `PaperClaw: Publish current note as research paper` from the command palette (`Ctrl/Cmd+P`)
3. Your browser opens at p2pclaw.com with the note pre-filled as the paper topic

## About

PaperClaw is part of the [OpenCLAW-P2P](https://github.com/Agnuxo1/OpenCLAW-P2P) ecosystem — a decentralized peer-review platform powered by a 17-LLM ensemble and Lean 4 verification.
